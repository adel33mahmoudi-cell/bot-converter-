# بوت تحويل الملفات

بوت تيليغرام متعدد الوظائف لتحويل الملفات بين الصيغ المختلفة.

## المميزات
- تحويل المستندات (PDF, DOCX, TXT, HTML)
- تحويل الصور (JPG, PNG, WEBP, GIF, ICO)
- تحويل الفيديو (MP4, AVI, MKV, MOV, GIF)
- تحويل الصوت (MP3, WAV, OGG, FLAC, M4A)

## طريقة الاستخدام
1. أرسل الملف للبوت
2. اختر الصيغة المطلوبة من الأزرار
3. انتظر لحظات واستلم الملف المحول

## النشر على Railway

1. انسخ هذا المشروع إلى GitHub
2. أنشئ حساب على [Railway](https://railway.app)
3. أنشئ مشروع جديد واربطه بالمشروع
4. أضف المتغيرات التالية:
   - `TOKEN`: مفتاح البوت من @BotFather

## المتغيرات المطلوبة
- `TOKEN`: مفتاح البوت (إجباري)
- `PORT`: المنفذ (اختياري، الافتراضي 8080)

## المتغيرات البيئية
| المتغير | الوصف | إجباري |
|---------|-------|--------|
| TOKEN | مفتاح البوت من BotFather | نعم |
| PORT | منفذ التشغيل (افتراضي 8080) | لا |
